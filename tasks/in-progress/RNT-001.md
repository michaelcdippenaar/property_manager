## Goal
Verify the rental mandate e-signing flow works cleanly for all 4 mandate types (Full Management, Letting Only, Rent Collection, Finders Fee) end-to-end, and fix any friction or bugs found.

## Acceptance criteria
- [ ] All 4 mandate types can be created, sent, signed, and finalized without error
- [ ] Owner + agent + (where applicable) property-manager signer flows verified
- [ ] Audit trail populates correctly for each signature
- [ ] Signed PDF generated via Gotenberg, stored on RentalMandate
- [ ] Email notifications fire to all parties at the right moments
- [ ] Any bugs found are fixed (or split into child tasks if out of scope)
