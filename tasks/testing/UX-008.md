## Goal

Add a minimum viable move-in prep checklist to the lease detail view so agents have an in-app workflow for keys handover, utilities notification, tenant app invite, and welcome pack after a lease is signed.

## Acceptance criteria

- [x] `content/product/features.yaml`: `tenant_onboarding.status` is corrected to reflect actual built state (set to `IN_PROGRESS` or `PLANNED` with `key_files: []` until this task ships)
- [x] A "Move-in prep" section appears on the lease detail view once the lease status is `active`
- [x] Checklist items (at minimum): Keys handed over, Utilities notified, Tenant app invite sent, Welcome pack delivered
- [x] Checklist state is persisted per lease (new `move_in_checklist` JSON field on Lease model or dedicated MoveInChecklist model)
- [x] Each checklist item records who completed it and when (agent name + timestamp)
- [x] The checklist is visible to the agent and read-only to the owner in the owner portal
- [x] `incoming_inspection` feature remains PLANNED; a placeholder card with "Coming soon" is shown below the checklist
- [x] Migration included; no data loss on existing leases

## Files likely touched
