## Goal
Write the first 10 help centre articles covering the questions a new agent is most likely to hit in their first 30 days.

## Acceptance criteria
- [x] 10 articles in `content/onboarding/help/` — one `.md` file each, slugified filename
- [x] Each article: title, 1-sentence summary, step-by-step with screenshots (describe where to place them), related links, last-updated date
- [x] Covers: how to create a mandate, how to invite a tenant, how to generate a lease, how to send for signing, what happens if the tenant doesn't sign, how to log a maintenance issue, how to see rent status, how to renew a lease, how to add a supplier, how to update owner dashboard permissions

- [x] Index file at `content/onboarding/help/README.md` listing all 10
