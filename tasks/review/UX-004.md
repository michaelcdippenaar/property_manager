## Acceptance criteria

- [x] Sequenced to the 15-step lifecycle (first 8 steps especially)
- [x] Copy is short (<15 words per tooltip), one idea per mark
- [x] Dismiss + "don't show again" states defined
- [x] Handoff includes a spec for the frontend dev (RNT task for implementation) — write the spec in the UX file, don't open the frontend task yet
