## Goal
Add the Android Capacitor platform to the agent app so an APK/AAB can be produced and Android smoke tests (QA-004) can be executed.

## Acceptance criteria
- [ ] `@capacitor/android` added to `agent-app/src-capacitor/package.json` matching `@capacitor/ios` version (`^6.2.1`)
- [ ] `npx cap add android` run; `android/` directory committed to repo
- [ ] `android/app/src/main/AndroidManifest.xml` reviewed — only `INTERNET` permission declared; no camera/location/contacts/phone-state
- [ ] `targetSdkVersion 34` and `minSdkVersion 23` confirmed in `build.gradle`
- [ ] App builds and launches on Android 14 emulator (Pixel 7 AVD)
- [ ] App builds and launches on Android 12 emulator (low-end profile AVD)
- [ ] `android:allowBackup="false"` set (see RNT-SEC-029 — can be resolved together)
