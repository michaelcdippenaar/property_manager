## Goal
Add the Android platform to the Klikk Agent Capacitor app so the app can be built and smoke-tested on Android devices, unblocking QA-004 and the Play Beta milestone.

## Acceptance criteria
- [x] `@capacitor/android` added to `agent-app/src-capacitor/package.json` matching the version of `@capacitor/ios` (currently `^6.2.1`)
- [x] `npx cap add android` run; `android/` directory committed
- [x] `android/app/src/main/AndroidManifest.xml` reviewed — only `INTERNET` permission declared; no camera/location/contacts/phone-state
- [x] `targetSdkVersion 34` and `minSdkVersion 23` confirmed in `build.gradle`
- [ ] App builds and launches successfully on Android 14 emulator (Pixel 7 AVD)
- [ ] App builds and launches successfully on Android 12 emulator (low-end profile AVD)
- [ ] No regressions on the existing iOS build

## Files likely touched
- `agent-app/src-capacitor/package.json`
- `agent-app/src-capacitor/android/` (new directory)
- `agent-app/src-capacitor/capacitor.config.ts` (may need android-specific config)
