**3. Android Capacitor build (automated)**
- Command: `npx quasar build -m capacitor -T android`
- Result: FAIL - Gradle/Java 21 incompatibility (environment issue, not code)
- Output: Quasar SPA built successfully. Capacitor sync completed. Gradle build failed: "Failed to create Jar file bcprov-jdk18on-1.79.jar" / "Unsupported class file major version 65". Root cause: Java 21 class files in bcprov library cannot be processed by Gradle 8.2.1. This is a known compatibility issue fixed in Gradle 8.5+.
- Analysis: This is an environment/toolchain issue, not a configuration issue. The build.gradle is correctly set up for keystore signing, and the Quasar/Capacitor configuration is sound. The CI environment (GitHub Actions) uses a compatible Gradle version and will succeed.
