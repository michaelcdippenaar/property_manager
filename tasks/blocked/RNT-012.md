## Goal
Add the required Apple privacy manifest to the Klikk Agent iOS app so TestFlight and App Store submission are not rejected for missing privacy declarations.

## Acceptance criteria

- [x] Declares `NSPrivacyAccessedAPICategoryFileTimestamp` (Capacitor file plugin accesses file timestamps)
