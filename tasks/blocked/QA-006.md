## Goal
Verify that every view in the admin SPA uses the Klikk design tokens consistently (Navy, Accent, fonts, spacing scale) and snapshot the current UI as a baseline for visual-regression tests going forward.

## Acceptance criteria
- [ ] Token audit report: every hardcoded colour hex not in `src/tokens/*` flagged and replaced
- [ ] Typography: every non-token font-size/weight flagged
- [ ] Spacing: every magic margin/padding normalised to token scale
- [ ] Storybook (or `/components` preview) for atoms + molecules + key organisms
- [ ] Chromatic / Percy / Playwright snapshot baseline captured per view on staging viewport
- [ ] Tie in to `klikk-design-standard` and `klikk-design-frontend-taste` skills
