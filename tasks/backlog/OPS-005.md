## Goal
Replace every plaintext / default-styled email with a branded, mobile-friendly, accessible HTML template — content sourced from `content/emails/` so copy edits don't require code changes.

## Acceptance criteria
- [ ] Template engine: MJML or maizzle → responsive HTML compiled at build
- [ ] Base layout: Klikk brand (Navy #2B2D6E, Accent #FF3D7F, Bricolage Grotesque fallback → system serif; DM Sans fallback → system sans), footer with unsubscribe + info officer contact
- [ ] Templates for: welcome (agent + tenant), mandate sent, mandate signed, lease sent for signing, lease signed, rent reminder, rent received, rent overdue, maintenance update, invite tenant, password reset, 2FA enrollment, 2FA recovery code reissue
- [ ] Plaintext fallback for every HTML
- [ ] Content sourced from `content/emails/*.md` (markdown + frontmatter)
- [ ] Accessibility: alt text on images, sufficient contrast, no text-in-image
- [ ] Deliverability: SPF + DKIM + DMARC set on sender domain; warmup schedule documented

## Files likely touched
- `content/emails/*.md` (new directory)
- `backend/apps/notifications/email_templates/*.mjml`
- `backend/apps/notifications/services/email.py`
- `docs/ops/email-deliverability.md`
