## Goal
Finalise the production domain setup so Klikk is reachable at stable URLs with valid certificates and properly configured email.

## Acceptance criteria
- [ ] Apex: `klikk.co.za` (marketing site)
- [ ] App: `app.klikk.co.za` (admin SPA)
- [ ] Tenant: `tenant.klikk.co.za` (tenant web)
- [ ] API: `api.klikk.co.za` (backend)
- [ ] All certs via Caddy auto-HTTPS, renewal automatic, monitored
- [ ] Email sender domain: `mail.klikk.co.za` with SPF, DKIM, DMARC `p=quarantine` (move to `reject` after 30 days)
- [ ] CDN for static assets (Cloudflare or S3 + CloudFront) with cache headers
- [ ] HSTS preload submission (after stable cert)
- [ ] Uptime monitors hitting each surface
