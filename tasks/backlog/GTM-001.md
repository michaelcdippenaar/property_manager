## Goal
Produce an ideal customer profile (ICP) and 2-3 buyer personas for Klikk Rentals v1 in the South African market. Ground in existing competitive research, not hypothetical.

## Acceptance criteria
- [ ] One ICP document in `content/sales/icp.md` covering firmographics, portfolio size, geographic focus, tech readiness, buying triggers
- [ ] 2-3 persona sheets in `content/sales/personas/` (e.g. small-portfolio landlord, rental agent at boutique agency, property manager at medium firm)
- [ ] Each persona: goals, pains, current tools, objections, decision criteria, day-in-the-life
- [ ] Segment sizing (rough, SA-specific) for each ICP tier
- [ ] Cross-referenced with `content/brand/` competitive intel
