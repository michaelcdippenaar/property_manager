## Goal
Create the required `PrivacyInfo.xcprivacy` privacy manifest so the iOS agent app is not rejected by App Store Connect during TestFlight or App Store submission.

## Acceptance criteria
- [ ] `PrivacyInfo.xcprivacy` created at `agent-app/src-capacitor/ios/App/App/PrivacyInfo.xcprivacy`
- [ ] Declares `NSPrivacyAccessedAPICategoryUserDefaults` (Capacitor uses NSUserDefaults for preferences)
- [ ] Declares `NSPrivacyAccessedAPICategoryFileTimestamp` (Capacitor file plugin accesses file timestamps)
- [ ] Privacy manifest added to Xcode project target membership (edit in Xcode or via `project.pbxproj`)
- [ ] `xcrun altool` upload or Xcode Organizer archive shows zero privacy manifest warnings

## Files likely touched
- `agent-app/src-capacitor/ios/App/App/PrivacyInfo.xcprivacy` (new)
- `agent-app/src-capacitor/ios/App/App.xcodeproj/project.pbxproj`

## Test plan

- Archive build in Xcode → Distribute App → confirm no privacy manifest warnings in App Store Connect validation
