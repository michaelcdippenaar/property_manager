## Goal

Complete the typography pass started in RNT-025 by closing the half-bump leaks, restoring H1 hierarchy (33% gap compressed), and eliminating all sub-WCAG-AA font sizes across the admin app.

## Background

RNT-025 bumped the Tailwind small tier (12/13/15px) but missed body default, `.page-title`, hard-coded px values in CSS files, and 10–11px WCAG violations in AppLayout.vue. MC reports the app still feels both small AND empty. Audit by Opus identified the root cause: hierarchy compression — H1 is only 33% bigger than body, with no visual anchor.

## Acceptance criteria

- [ ] **1. Tailwind config** (`admin/tailwind.config.js`) — extend fontSize tier:
  ```js
  fontSize: {
    micro: ['12px', { lineHeight: '17px' }],
    xs:   ['13px', { lineHeight: '18px' }],
    sm:   ['15px', { lineHeight: '22px' }],
    base: ['16px', { lineHeight: '25px' }],
    lg:   ['18px', { lineHeight: '28px' }],
    xl:   ['22px', { lineHeight: '30px' }],   // was 20 → H1 anchor
    '2xl':['28px', { lineHeight: '34px' }],   // was 24 → stat values
    '3xl':['34px', { lineHeight: '40px' }],
  }
  ```
- [ ] **2. `admin/src/assets/main.css`**:
  - body `font-size: 16px` (was 15px)
  - `.page-title { @apply text-2xl font-bold ... }` (was text-xl)
  - Fix `.text-micro` alias: `@apply text-[12px] text-gray-500` (currently does `@apply text-xs` — token drift)
- [ ] **3. `admin/src/styles/klikk-components.css`**: replace every hard-coded `font-size: 13px` with `14px`; bump `12px → 13px`; lift `.klikk-page-header h1` to 30px.
- [ ] **4. `admin/src/assets/agent-shell.css`**: apply 12→13 / 13→14 / 14→15 sweep on `.tab`, `.btn`, `.ev-row`, `.search input`, `.badge`, `.ledger td`. Scoped — safe.
- [ ] **5. WCAG AA fixes** — search & replace:
  - All `text-[10px]` → `text-[11px]` minimum (5 occurrences) — under WCAG floor
  - In `admin/src/components/layout/AppLayout.vue`: bump 11 occurrences of `text-[11px]` → `text-xs` (13px) for sidebar labels
  - `tiptap-editor.css:43,148` 10px → 12px
  - `klikk-badge.size-sm` 11px → 12px
- [ ] **6. Contrast sweep** — replace hot `text-gray-400` on white → `text-gray-500` (currently 4.4:1, fails AA 4.5:1 body) at:
  - `OnboardingItem.vue:23`
  - `MoveInChecklist.vue:7,86`
  - `OverdueMaintenanceWidget.vue:35`
  - `PropertyInformationEditor.vue:25`
  - `LeaseTimelineGantt.vue:8`
- [ ] **7.** `cd admin && npm test` — no new regressions vs RNT-025 baseline (8 pre-existing failures in `signature-block-signing.browser.test.ts` are acceptable).
- [ ] **8.** `cd admin && npx vite build` — must succeed.
- [ ] **9.** Add a Vitest test asserting Tailwind exposes `xl: 22px`, `2xl: 28px`, body 16px (snapshot of computed style on a mounted component).
