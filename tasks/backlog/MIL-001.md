## Goal
The single go/no-go gate for taking on the first paying customer. This task is complete when every item in the checklist below is checked, and the first agency can be onboarded end-to-end without a developer in the loop.

## Acceptance criteria — launch gate checklist

### Security & compliance
- [ ] Secrets rotated + git history clean (RNT-SEC-001)
- [ ] Rate limits live on public endpoints (RNT-SEC-002)
- [ ] 2FA enforced for staff/agency_admin/agent/owner (RNT-SEC-003)
- [ ] Test endpoints removed from prod (RNT-SEC-004)
- [ ] RHA compliance gate blocking (RNT-SEC-007)
- [ ] Legal pages published + sign-up consent capture (OPS-004)

### Ops readiness
- [ ] CI/CD green and deploying to prod (OPS-001)
- [ ] Sentry live across all surfaces (OPS-002)
- [ ] Backups running + restore drill passed once (OPS-003)
- [ ] Production domain + HTTPS + email deliverability (OPS-006)
- [ ] Uptime monitoring + on-call alerting (OPS-008)
- [ ] Production secrets vault in use (OPS-009)

### Quality
- [ ] Backend regression suite green (QA-001)
- [ ] Agent golden path E2E green (QA-002)
- [ ] Tenant golden path E2E green (QA-003)
- [ ] RBAC matrix passes (QA-009) — no cross-agency / cross-tenant leaks
- [ ] Website QA (QA-010)

### Product
- [ ] Tenant onboarding UI complete (RNT-QUAL-001)

### Onboarding
- [ ] Agent first-cycle audit complete with priority fixes landed (UX-001)
- [ ] Tenant first-cycle audit complete (UX-002)
- [ ] Welcome video scripts finalised (UX-003)

### Go-to-market
- [ ] Target market + ICP + positioning locked (GTM-001, GTM-002)

### Onboarding dry-run
- [ ] Complete a scripted dry-run of onboarding a fictitious agency end-to-end: signup → invite agents → create first property → create first mandate → invite first tenant → sign lease → receive first rent payment → close first maintenance ticket
- [ ] No developer intervention required
- [ ] Dry-run transcript + any last-mile friction logged in `docs/launch/first-client-dry-run-2026-04.md`
