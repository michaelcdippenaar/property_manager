## Goal
Add a `compile_email_templates.sh` script (or Makefile target) that re-inserts Django template conditional blocks (`{% if cta_url %}`, `{% if note %}`) after running `npx mjml`, so recompiling `base.mjml` does not silently break email rendering.

## Acceptance criteria
- [x] `scripts/compile_email_templates.sh` (or `Makefile` target) runs `npx mjml base.mjml -o compiled/base.html` then re-patches the conditional rows
- [x] Script is documented in `README` / `CONTRIBUTING` — "always use the script, not raw `npx mjml`"
- [x] Running the script produces a `compiled/base.html` that passes the existing email rendering smoke test (CTA button only renders when `cta_url` is provided)

## Files likely touched
- `scripts/compile_email_templates.sh` (new)
- `Makefile` (optional target)
- `backend/apps/notifications/email_templates/base.mjml`
- `backend/apps/notifications/email_templates/compiled/base.html`

## Test plan
**Manual:**
- Modify a branding detail in `base.mjml`, run the script, confirm compiled output retains `{% if cta_url %}` blocks and CTA renders conditionally

**Automated:**
- Existing email send smoke tests pass after recompile
