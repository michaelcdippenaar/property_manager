## Goal
Create `PrivacyInfo.xcprivacy` for the Klikk Agent iOS app declaring all accessed privacy-sensitive APIs so App Store Connect does not reject the build.

## Acceptance criteria
- [x] `PrivacyInfo.xcprivacy` created at `agent-app/src-capacitor/ios/App/App/PrivacyInfo.xcprivacy`
- [x] Declares `NSPrivacyAccessedAPICategoryUserDefaults` (Capacitor uses NSUserDefaults for preferences)
- [x] Declares `NSPrivacyAccessedAPICategoryFileTimestamp` (Capacitor file plugin accesses file timestamps)
- [ ] Privacy manifest added to Xcode project target membership
- [ ] Build passes App Store Connect upload validation (`xcrun altool` or Xcode Organizer shows no privacy manifest warnings)
