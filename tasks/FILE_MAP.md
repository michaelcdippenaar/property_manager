# Klikk FILE MAP — Static Area → Files Index

## How to use

**PM (rentals-pm):** When authoring a new task, look up the area in the table below and paste the matching rows into `Files likely touched`. This removes the need for implementers to grep the codebase before starting.

**Autopilot (klikk-rentals-autopilot):** Before claiming a task, cross-check `Files likely touched` against this map. If two in-flight tasks share a canonical file, flag the overlap to PM before starting the second.

**Implementers:** Trust the `Files likely touched` field in your task brief — it was pre-loaded from this map. If you discover a missing file, update the relevant row here and leave a note in your handoff.

**Out-of-scope areas:** `the_volt/` (Vault33 product) and `website/` (Astro marketing site) are listed for reference but are outside the Rentals v1 task queue. Do not assign RNT/QA tasks to those paths.
